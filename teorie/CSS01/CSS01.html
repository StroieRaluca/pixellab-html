Cascading Style Sheets

3 tipuri de a-l folosi:
  • Inline Styles - Ca atribut pe HTML
  • Internal Styles - In fisierul HTML folosind <style>
  • External Styles - In fisier dedicat, folosind <link href=""/>

Prioritatea

In ordinea specificata mai sus, CSS-ul se va suprascrie datorita "specificitatii" in felul urmator: 1. Cel mai putin specific este CSS-ul extern. Asta pentru ca un fisier extern poate fi folosit de catre mai multe documente HTML.
2. Urmatoarea varianta, mai specifica, este CSS-ul intern, cel declarat in fisier. Asta pt ca se refera doar la HTML-ul curent, si la continutul lui.
3. Cea mai specifica varianta este CSS-ul inline, acesta se aplica (si refera) la un singur element.

Reguli CSS

Regulile CSS sunt felul in care specificam ce valori vrem sa dam caror proprietati de pe care elemente.

  1. Primul pas este scrierea selectorului.
  2. Al doilea pas este setul de proprietati pe care dorim sa le modificam, intre acolade si separate prin punct virgula.

Selectori CSS

Selectorii sunt anumite bucati de text, care vor avea rolul de a reprezenta "adresa" elementelor pe care vrem sa le selectam, pentru modificare.

Tipuri de selectori

  • Selectori Tag / Element name - "h1" va selecta elementele de tip h1.
  • Selectori de tip class - .lorem va selecta toate elementele cu clasa "lorem". Totul porneste de la punct.
  • Selectori de tip ID - #ipsum va selecta primul element cu ID-ul ipsum.
    ASDF
  • Selectori de tip atribut - Acei selectori pe care-i putem folosi pentru a selecta elemente in functie de orice atribut.

    Sus la prioritati. Sus la titlu.

    Sintaxa de baza a selectorilor pe baza de atribute este paranteze drepte pt incapsularea selectorului [], in care specificam ce atribut cautam, ce operator de comparatie folosim, si respectiv ce valoare vom compara.

Toti selectorii pot fi combinati intre ei, atat prin:
- fuziune (lipire), de exemplu h1.principal - acesta va reprezenta un element de tip "h1" care are clasa "principal"
- pauza (pauza intre 2 selectori) - aceasta pauza va insemna o combinatie de tip ierarhic, de exemplu "a b" va insemna un element de tip "b" care se afla intr-un element de tip "a"

Selectori complecsi: ierarhizare

  • Descendent: "a b" cu pauza va selecta orice element b din a, indiferent daca e descendent direct sau nu.
  • Descendent direct: "a>b" cu > va selecta toate elementele b care sunt descendentii directi ai unui element a
  • Sibling: "a ~ b" cu ~tilda va selecta orice b care urmeaza dupa un a in pagina
  • Direct sibling: "a + b" cu plus va selecta orice b care este pozitionat imediat dupa un a, pe acelasi nivel

Combinarea mai multor selectori

Daca dorim sa aplicam o regula CSS pe mai multi selectori, si anume nu pe un selector mai specific / complex, ci pur si simplu pe 2 selectori diferiti sau mai multi, putem sa-i separam folosind virgula.

ex: a, p { color: green }; <- va selecta atat linkurile cat si paragrafele

Culorile in CSS

Culorile in CSS (sau alte arii ale informaticii) pot fi reprezentate in mai multe feluri, dar totul porneste de la ideea e pixel, si felul in care se combina lumina emisa de un pixel pentru a forma o anumita nuanta.

  • Culori text: Acestea specifica valori de tipul "red". Acestea nu sunt recomandate pentru ca diferite browsere pot interpreta culorile prin diferite nuante.
  • RGB - red green blue sub forma "rgb(R,G,B)" unde R,G,B sunt valori numerice pentru intensitatea fiecarei culori. Aceste intensitati vor fi in intervalul 0-255, de exemplu rgb(255, 0, 255) => va rezulta violet.
  • HEX- Hex code, se refera la un cod format din 6 caractere care de fapt reprezinta 2 perechi de caracter, FF00FF - va fi violet, pentru ca de fapt se refera tot la RGB, sub forma "FF" "00" "FF"
  • HSL - Hue saturation lighting, un sistem folosit mai rar.

Primele proprietati CSS

color - va reprezenta culoarea textului dintr-un element

border - va reprezenta un chenar. El va primi o valoare sub forma "_A_px _B_ _C_" unde _A_ reprezinta un nr de pixeli care va da grosimea chenarului, _B_ tipul sau aspectul chenarului, exemplu solid, dotted, dashed, si _C_ va reprezenta culoare, fie ea scrisa rgb, hex, sau cu nume.
Exemplu: a { border: 1px solid blue };

background-color - va reprezenta culoarea fundalului unui element
